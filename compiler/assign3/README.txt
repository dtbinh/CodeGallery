Implementation details:

This program implements a single dimension array which supports the 
following operations:

1. Declare and define an array:
	e.g. a[] = {1,2,3,4,5};
2. Access an element in array:
	e.g. print a[i];
3. Assign value to array:
	e.g. a[i] = b + c;
4. Read into array:
	e.g. read a[i];

The index of array can be integers, array elements, and other expressions.

******************************************************************************
Test files:
Four files have been provided to test the program:
1. average.sc: calculate the average of numbers in array
2. max.sc: calculate the maximum element in array
3. sort.sc: sort the given array 
4. count.sc: count the frequency of each number

******************************************************************************
How to run the program:
A script has been provided to run the program, simple use the following 
command:
	./run.sh FILENAME
	e.g. ./run.sh max
